This folder contains the code for Arithmetic Coding algorithm that has been implemented in Java. To run the code follow the following steps:

1. The program uses "arithmetic_input.txt" as the input file. Put the text that has to be compressed here.

2. The program produces "arithmetic_output.txt" which is the compressed file.

To run the code type the following commands from the windows command prompt in the given order:

javac Arithmetic_coding.java
java Arithmetic_coding
