This folder contains the code for asymptotically optimal LZ77 algorithm that has been implemented in Java. To run the code follow the following steps:

1. The program uses "lz77.txt" as the input file. Put the text that has to be compressed here.

2. The program produces "output_lz77.txt" which is the compressed file.

To run the code type the following commands from the windows command prompt in the given order:

javac Lz77.java
java Lz77
